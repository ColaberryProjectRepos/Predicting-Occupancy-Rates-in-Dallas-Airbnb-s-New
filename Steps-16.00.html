Adding project logo's and Title 16


Adding Colaberry Logo:  

1.     Insert: Image and select Colaberry logo from the downloaded file

2.     Go to General tab and set Properties: Size: Height = 92, Width = 276

                                                Position: Horizontal = 0, Vertical = 0

Adding Airbnb Logo:

1.     Insert: Image and Select Airbnb Logo from the downloaded file.

2.     Go to General tab and set Property: Size: Height = 92, Width = 154

                                                Position: Horizontal = 1125, Vertical = 0

Adding Title:

1.     Insert a Text Box and insert ”Predicting Occupancy Rates in Dallas Airbnb’s”

2.     The Font and style will be set to Segoe UI, 28

3.     Go to General tab and set Property: Size: Height = 92, Width = 850

                                                Position: Horizontal = 276 Vertical = 0
