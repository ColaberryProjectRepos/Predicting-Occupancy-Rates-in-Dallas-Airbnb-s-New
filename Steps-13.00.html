Creating Calculated Column 13


1.     Make a right click on FinalFabricFile table

2.     Select New column

3.     Start writing the Dax formula as follows:

Season =

SWITCH (

    TRUE (),

    AND (

        FinalFabricFile[Date_x] >= DATE ( YEAR ( FinalFabricFile[Date_x] ), 3, 20 ),

        FinalFabricFile[Date_x] <= DATE ( YEAR ( FinalFabricFile[Date_x] ), 6, 20 )

    ), "Spring",

    AND (

        FinalFabricFile[Date_x] >= DATE ( YEAR ( FinalFabricFile[Date_x] ), 6, 21 ),

        FinalFabricFile[Date_x] <= DATE ( YEAR ( FinalFabricFile[Date_x] ), 9, 22 )

    ), "Summer",

    AND (

        FinalFabricFile[Date_x] >= DATE ( YEAR ( FinalFabricFile[Date_x] ), 9, 23 ),

        FinalFabricFile[Date_x] <= DATE ( YEAR ( FinalFabricFile[Date_x] ), 12, 20 )

    ), "Fall",

    OR (

        FinalFabricFile[Date_x] >= DATE ( YEAR ( FinalFabricFile[Date_x] ), 12, 21 ),

        FinalFabricFile[Date_x] <= DATE ( YEAR ( FinalFabricFile[Date_x] ) + 1, 3, 19 )

    ), "Winter",

    BLANK()

)


4.     Click the commit button to save it.
