Creating Occupancy Rate by Neighbourhood Map 24


In order to create the map visual we need to add a new visual “iconMapV3”.

Step 1. Download a new visual

1.     Go to Visualization tab
2.     Get more visual
3.     Search iconMapV3
4.     Download and add to your visuals

Step 2. Creating a map by using our .geojson file

·       Go to google and search for geojson.io
·       Open .geojson file in not pad
·       Copy all and past it on the above website
·       The map will be created for us
·       Save the map to your machine

Step 3. Store and host our geojson map

·       On a google search GITHUB
·       Create a new trial account
·       Create a new repository
·       Upload an existing file
·       After review select Row
·       Copy and save the weblink from the top

Step 4. Visualization

·       Select iconMapV3 visual 
·       Category: neighborhood_x
·       Size: OccupRate
Formatting: Fill color = fx( Minimum = F6F69A,  Center = 606423, Maximum = 000000)


Visual: GeoJSON = ON, Copy the URL that you copy

General: Property: Size: Height = 343, Width = 501,

                                          Position: Horizontal = 774, Vertical = 379

Title: ON, Title = Occupancy Rate By Neighbourhood, Font = DIN, 14, Text Color = White (#FFFFFF), Background Color = (#E68F96), Horizontal Alignment = Center

Effects: Background = ON, Color = White(#FFFFFF), Transparency = 0%

                  Visual Boarder = ON, Color = Grey(#E6E6E6)

Tooltips = ON
